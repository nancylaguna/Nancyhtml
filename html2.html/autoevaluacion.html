2.1 Indique si cada uno de los siguientes es verdadero o falso, explique por qué

a) Una lista ordenada no puede anidarse dentro de una lista desordenada. ------ Falso - La lista ordenada puede anidarse dentro de una lista desordenada.

b) El elemnto br representa un salto de linea. ------ Verdadero.

c) Los hipervínculos se denotan mediante elementos link. ------ Falso - Los hipervínculos se denotan mediante elementos a.

d) La anchura de todas las celdas de datos en una tabla debe ser igual. ------ Falso - Se puede especificar la anchura de cualquier columna.

e) Estamos limitados a un máximo de cinco vínculos internos por página. ------ Falso - Se pueden tener numeros ilimitados de vínculos internos.
